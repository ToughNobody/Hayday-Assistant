# 卡通农场小助手


## 简介

卡通农场小助手是一款基于Autox.js,专为《卡通农场》(Hay Day) 国际服游戏玩家设计的自动化辅助工具。通过该工具，玩家可以实现农场的自动化管理，包括种植、收获、售卖等操作，大大提高游戏效率。

## 功能特性

### 🌾 刷地功能
- 支持多种作物种植
- 可自定义作物成熟时间

### 📦 仓库售卖
- 自动统计仓库物品数量
- 当仓库空间不足时自动售卖多余物品
- 可设置售卖触发阈值和目标数量

### 🔄 账号切换
- 支持多账号管理
- 自动切换账号功能
- 支持邮箱账号和存档账号两种方式

### 📊 数据统计与推送
- 仓库物品统计功能
- 支持多种推送方式（pushplus、Server酱、WxPusher）
- 统计结果可通过消息推送给用户

### 🎯 汤姆功能
- 可在刷地中跑汤姆，也可单独使用
- 支持多账号跑汤姆

详细的使用教程请参考：[使用文档](https://docs.qq.com/doc/DWEtDUXB0U0dISGxo)

## 配置说明

### 权限设置
- **无障碍服务**：用于模拟点击操作
- **截图权限**：用于图像识别
- **悬浮窗权限**：显示悬浮提示文字窗口

## 注意事项

1. 使用设备分辨率需为720*1280，DPI320
2. 游戏语言需设置为简体中文
3. 使用前请确保已授予所有必要权限
4. 本工具仅供学习交流使用，请遵守游戏官方规则
5. 使用过程中如遇到问题，请查看日志文件或联系开发者
